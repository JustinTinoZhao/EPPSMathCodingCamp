Quarto Coding

Quarto is an expanded version of R Markdown for rendering reports with integrated code like the one you are currently reading. To create a quarto document, select Create a new Quarto document from the New File drop down in RStudio.

As this is a markdown style, most of the options this gives you access to are ways to modify text or add other embellishments to your report. Below are a few examples on how to change the text.

LaTeX

LaTeX is a high-quality typesetting system that is widely used for producing scientific and mathematical documents. It is especially useful when your report includes complex equations, tables, and citations.

To create a LaTeX document, you typically start with a .tex file and compile it into a PDF using a TeX engine such as pdflatex, xelatex, or lualatex.

While Quarto is excellent for combining code and text, LaTeX is the go-to tool when you want full control over formatting in academic writing, especially for papers with complex equations.

Below are examples of common LaTeX syntax:

Document Structure

A minimal LaTeX document looks like this:

\documentclass sets the type of document.

adds support for math symbols.

prints title, author, date.

\begin{document} is the start of visible content.

\documentclass{article}
\usepackage{amsmath} % For math symbols
\title{My First LaTeX Document}
\author{Your Name}
\date{\today}

\begin{document}

\maketitle

Hello, this is my first LaTeX document.

\end{document}

Math in LaTeX

Inline math uses ( … ) or $ … $

Displayed math uses [ … ] or \[ ... \]

Math rendering in Quarto supports LaTeX syntax through MathJax

LaTeX Syntax Output / Description
\( E = mc^2 \) Inline math (within a sentence)
\[\int_a^b x^2\,dx = \frac{b^3 - a^3}{3} \] Displayed math (centered formula)
x_i^2 Subscript & superscript
\frac{a}{b} Fraction: a over b
\sum_{i=1}^n i^2 Summation

Figures in LaTeX

Image file must be in the same folder as .tex or give full path.

Common formats supported: .png, .jpg, .pdf, .eps

LaTeX Syntax Output / Description
\usepackage{graphicx} Add in preamble to enable image handling
\begin{figure}[h] ... \end{figure} Inserts a figure block
\includegraphics[width=0.5\textwidth]{image.png} Embeds image scaled to 50% text width
\caption{Sample Image} Adds a caption below the figure
\centering Centers the figure

Citations and Bibliography in LaTeX

A .bib file is just a text file containing entries like:

  author = {Smith, John},
  title = {Sample Title},
  journal = {Journal Name},
  year = {2022}
}
LaTeX Syntax Output / Description
\usepackage{biblatex} Enables BibLaTeX reference management
\addbibresource{references.bib} Adds a .bib file to the document
\cite{key} In-text citation (e.g., [Smith, 2022])
\printbibliography Prints the bibliography section

Compiling LaTeX Documents

Most editors (e.g., Overleaf, VS Code, TeXstudio) automate this for you.

Compilation is necessary to generate the PDF output.

Tool / Command Purpose / Description
pdflatex filename.tex Compiles .tex file into a PDF
xelatex filename.tex Unicode-friendly compiler (good for multilingual)
lualatex filename.tex LaTeX engine based on Lua scripting
bibtex filename.aux Compiles bibliography (if using BibTeX)
latexmk -pdf filename.tex Automatically compiles and resolves dependencies

# LaTeX

------------------------------------------------------------------------

LaTeX is a high-quality typesetting system that is widely used for producing scientific and mathematical documents. It is especially useful when your report includes complex equations, tables, and citations.

To create a LaTeX document, you typically start with a `.tex` file and compile it into a PDF using a TeX engine such as `pdflatex`, `xelatex`, or `lualatex`.

While Quarto is excellent for combining code and text, LaTeX is the go-to tool when you want full control over formatting in academic writing, especially for papers with complex equations.

Below are examples of common LaTeX syntax:


## Document Structure

------------------------------------------------------------------------

A minimal LaTeX document looks like this:

\documentclass sets the type of document.

\usepackage{amsmath} adds support for math symbols.

\maketitle prints title, author, date.

\begin{document} is the start of visible content.

```latex
\documentclass{article}
\usepackage{amsmath} % For math symbols
\title{My First LaTeX Document}
\author{Your Name}
\date{\today}

\begin{document}

\maketitle

Hello, this is my first LaTeX document.

\end{document}
```


## Headings in LaTeX

------------------------------------------------------------------------

| **LaTeX Syntax**        | **Output** (Document Section) |
|:------------------------|:------------------------------|
| `\section{Main Section}`     | Main (level 1) section heading |
| `\subsection{Subsection}`   | Subsection (level 2)            |
| `\subsubsection{Sub-sub}`   | Sub-subsection (level 3)        |


## Text Formatting in LaTeX

------------------------------------------------------------------------

| **LaTeX Syntax**            | **Output**            |
|:----------------------------|:-----------------------|
| `\textit{italic}`           | *italic*               |
| `\textbf{bold}`             | **bold**               |
| `\underline{underlined}`    | <ins>underlined</ins>  |
| `\texttt{monospace}`        | `monospace`            |


## Math in LaTeX

------------------------------------------------------------------------

Inline math uses \( ... \) or $ ... $

Displayed math uses \[ ... \] or $$ ... $$

Math rendering in Quarto supports LaTeX syntax through MathJax


| **LaTeX Syntax**                              | **Output** / Description           |
|:---------------------------------------------|:-----------------------------------|
| `\( E = mc^2 \)`                              | Inline math (within a sentence)    |
| `\[\int_a^b x^2\,dx = \frac{b^3 - a^3}{3} \]` | Displayed math (centered formula)  |
| `x_i^2`                                       | Subscript & superscript            |
| `\frac{a}{b}`                                 | Fraction: a over b                 |

## Tables in LaTeX

------------------------------------------------------------------------

## Citations and Bibliography in LaTeX

------------------------------------------------------------------------

A .bib file is just a text file containing entries like:
```@article{smith2022,
  author = {Smith, John},
  title = {Sample Title},
  journal = {Journal Name},
  year = {2022}
}
```


| **LaTeX Syntax**                              | **Output / Description**                            |
|:----------------------------------------------|:----------------------------------------------------|
| `\usepackage{biblatex}`                        | Enables BibLaTeX reference management               |
| `\addbibresource{references.bib}`             | Adds a `.bib` file to the document                  |
| `\cite{key}`                                   | In-text citation (e.g., [Smith, 2022])              |
| `\printbibliography`                           | Prints the bibliography section                     |


## Compiling LaTeX Documents

------------------------------------------------------------------------

Most editors (e.g., Overleaf, VS Code, TeXstudio) automate this for you.

Compilation is necessary to generate the PDF output.


| **Tool / Command**                        | **Purpose / Description**                          |
|:------------------------------------------|:---------------------------------------------------|
| `pdflatex filename.tex`                   | Compiles `.tex` file into a PDF                    |
| `xelatex filename.tex`                    | Unicode-friendly compiler (good for multilingual) |
| `lualatex filename.tex`                   | LaTeX engine based on Lua scripting                |
| `bibtex filename.aux`                     | Compiles bibliography (if using BibTeX)            |
| `latexmk -pdf filename.tex`               | Automatically compiles and resolves dependencies   |
